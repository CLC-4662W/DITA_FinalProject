Tips for using overrides.

This document gives tips about when to select which Reveal.js default override.

Overriding with new CSS code- Create new CSS code if you want the formats to apply only to a specific portion of the presentation, such as one slide.

Override by deleting the resets- The resets defined in the Reveal.js CSS file for the demonstration presentation are all class id's. Do not delete any of these class id's unless you're sure that you want all your slides to use the standard CSS defaults.

Revise existing CSS classes- Revising existing classes will apply your changes universally to all slides that use that class. This is a good way to make universal format changes, such as font size or centering/left justification.

  • Override with new CSS code
    It is possible to override Reveal.js defaults with new CSS code.
  • Override by deleting resets
    You can override Reveal.js defaults by deleting the resets in the CSS file.
  • Revise existing CSS classes
    You can override Reveal.js defaults by modifying the CSS classes.
