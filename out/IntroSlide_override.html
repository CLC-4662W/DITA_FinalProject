Remove/Reset Reveal.js defaults

In some cases the Reveal.js default formats will not be suitable for your purposes. There are several methods you can use to override these defaults.

Choose one or more methods to reset defaults:
  • Write new CSS code.
  • Delete resets for a specific element.
  • Modify CSS classes for .reveal and .slides.
  • Override with new CSS
    It is possible to override Reveal.js defaults with new CSS code.
  • Override by deleting resets
    You can override Reveal.js defaults by deleting the resets in the CSS file.
  • Revise existing classes
    You can override Reveal.js defaults by modifying the CSS classes.
