Override with new CSS

It is possible to override Reveal.js defaults with new CSS code.

  1. Create an id tag by adding the pound symbol before a name.
    For example, #TitleSlide
  2. Add CSS code for the desired effect to this id tag.
    For example, to center the text, {text-align: center}
  3. In your HTML file, create a div with this id
    div id="TitleSlide"

Every div in the HTML file with this id will be centered.
