Override by deleting resets

You can override Reveal.js defaults by deleting the resets in the CSS file.

  1. Find the reset classes under the "reset styles" comment
  2. Delete the class for the element you wish to have follow normal defaults.
    CAUTION:
    Be sure you know the standard CSS defaults before using this method. Also be sure this is how you want your slides to appear.
