Create a title slide

You can create a slide with centered headings for use as a title slide.

  1. Reset the Reveal.js CSS file such that this slide will have centered content.
    Note: This step will not be necessary if you didn't make any changes for the presentation to be left justified because the Reveal.js default is to center all of the slides.
  2. Enter text into h1, h2, and h3 elements to create a title slide.
